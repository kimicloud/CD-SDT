About Syntax-Directed Translation

What is SDT?

Syntax-Directed Translation extends context-free grammar by associating semantic rules with grammar productions, enabling tasks like type checking, code generation, and expression evaluation.

S-Attributed SDT

Uses only synthesized attributes with bottom-up evaluation. Perfect for shift-reduce parsers and ensures efficient implementation without inherited attributes.

L-Attributed SDT

Uses both synthesized and inherited attributes with left-to-right propagation. More flexible but requires careful dependency management.
